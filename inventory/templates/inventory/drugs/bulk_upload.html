Upload Guidelines

  • Required Columns: Column headers must exactly include: drug_name, category, unit, reorder_level, min_stock, max_stock
  • Valid Categories: Must match one of: TABLET, CAPSULE, SYRUP, INJECTION, OINTMENT, OTHER
  • Stock Limits: Numeric values (`reorder_level`, `min_stock`, `max_stock`) must be integers greater than or equal to 0.
  • Duplicate Check: If a drug with the same name already exists in the database, it will be skipped automatically without throwing errors.
  • All-or-Nothing Import: If any validation error is detected on any row, the entire upload is cancelled to prevent half-imported data.
